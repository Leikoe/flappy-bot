# flappy bot

a simple flappy bot ai, which uses a neural network to classify ingame screenshots and decide on key presses

> note: this repo was tested on https://flappybird.io/

## requirements

- numpy
- keyboard

**os specific packages**
for macos:
- tensorflow-macos
- tensorflow-metal
for linux/windows:
- tensorflow
